Servidor SOAP de Conversiones

Servicio listo para conversiones de Longitud, Masa y Temperatura.

WSDL: /UnitConversionService?wsdl (relativa al contexto de la aplicacion)

Flujo de uso

  1. Consumir la operación login con usuario y clave para validar acceso.
  2. Consumir convertLength, convertMass o convertTemperature sin token.

Credenciales quemadas

  • Usuario: MONSTER
  • Clave: MONSTER9

Unidades soportadas

Longitud: mm, cm, m, km, in, ft, yd, mi

Masa: mg, g, kg, lb, oz, t

Temperatura: c, f, k
